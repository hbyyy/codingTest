### 그래프 순회

- 그래프의 각 정점을 방문하는 과정
- BFS, DFS

```python
def DFS(v, discovered = []):
	discovered.append(v)
    for i in graph[v]:
        if i not in discovered:
            DFS(i, discovered)
            
def DFS(start_v):
    discovered = []
    stack = [start_v]
    
    while stack:
        v = stack.pop()
        if v not in discovered:
            discovered.append(v)
            for i in graph[v]:
                stack.append(i)
```
